h1. Declarative Authorization Rails 3 Example App

Sample Rails 3 and Devise app using "declarative_authorization":https://github.com/stffn/declarative_authorization gem.

Based on the Declarative Authorization RailsCast http://railscasts.com/episodes/188-declarative-authorization

h2. Installation

bundle install
rake db:create
rake db:migrate
rake db:fixtures:load

Then start the server, and you should be good to go!

h2. Development

* Data persistence: SQLite with ActiveRecord
* Template Engine: Haml
* Testing Framework: RSpec and Factory Girl
* Front-end Framework: Twitter Bootstrap (Sass)
* Authentication: Devise
* Authorization: Declarative Authorization
